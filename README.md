### git에 대한 규칙
1. 개인작업은 각자 branch를 만들어서 작업한다.
2. branch명은 "본인의 성"로 한다.
3. branch에서 작업 전에 본인이 작업 시작을 알린다, 알리지 않으면 다른 인원이 물어봐준다.
4. commit 시 메세지는 "변경사항"로 한다.
5. push는 main에서 branch를 merge한 후에 main에서 push한다.
6. push 하기 전에 팀원에게 알리고 한다.
